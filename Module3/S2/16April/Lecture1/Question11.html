CSS Box Model Demonstration

Box 1 (content-box)
Box 2 (border-box)
Box 3 (content-box)

Key Differences:

Box 1: Uses box-sizing: content-box (default). The width (200px) only applies to the content area. Padding (20px) and border (10px) are added to this width, making the total rendered width 260px (200 + 20 + 20 + 10 + 10).

Box 2: Uses box-sizing: border-box. The width (200px) includes content, padding (30px), and border (5px). The actual content width is 130px (200 - 30 - 30 - 5 - 5).

Box 3: Uses content-box with asymmetric padding (15px top/bottom, 40px left/right) and margin (10px top/bottom, 30px left/right). Notice how the dotted border affects the total size.
